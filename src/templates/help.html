Aide de Yogo

Introduction

Yogo est un site de gestion de projet fonctionnant sur le système du tableau 'TO-DO', 'DOING', 'DONE' (à la manière de trello ou kanboard).

Projets

Vous pouvez créer un projet en cliquant sur le bouton de la page d'accueil ou sur le bouton du menu déroulant en haut à droite.
Les projets inactifs sont inacessibles par les membres autres que les administrateurs
Les informations du projet se trouvent dans l'onglet projet (où vous pouvez aussi le supprimer) et vous pouvez changer ces informations dans l'onglet Modifier le projet.
Les membres s'ajoutent dans l'onglet Gérer les membres, c'est aussi dans cet onglet que sont gérés les administrateurs du projet

Les tâches

Vous pouvez retrouver l'ensemble des taches qui vous sont assignées dans l'onglet principal Tâches.
Vous pouvez retouver l'ensemble des tâches d'un projet dans l'onglet tâches de celui-ci.
Une tache peut être assignée à un utilisateur du projet, celui peut aussi la papsée (PAPS : Premier Arrivé Premier Servi) ou la dépapsée pour l'abandonner.
Une tache peut aussi avoir des prérequis, c'est à dire d'autres taches qui si elles ne sont pas réalisées, empêchent le changement de statuts des tâches dépendantes.
Une tâche peut aussi posséder un ou plusieurs tags

Les tags

Les tags permettent de mieux classer les tâches. Ils sont relatifs à un projet et peuvent être gérés depuis l'onglet Gérer les tags.

Administrateurs d'un projet

Les administrateurs d'un projet sont les seuls à pouvoir :
  • Supprimer le projet
  • Changer le titre, la description ou si le projet est actif
  • Retirer un membre du projet
Ils peuvent être nommés dans l'onglet Gérer les membres du projet.
Par défaut, le créateur d'un projet est administrateur.

Profil

Les informations de votre compte sont rassebmlées sur votre profil.
Vous y trouverez aussi vos préférences mails et télégrams.
